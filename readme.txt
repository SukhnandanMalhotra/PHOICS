DJANGO PHOICS APP
This is django based app

Features:-
- User can signup using his email-id
- He can reset his password
- He can login, if he already have account
- Photos can be uploaded
- Photos can be edited and re-edited
- He can make his photo either public or private
- He can edit his profile information on profile page and he can set his profile pic here
- He can search any user having account on phoics
- There is a newsfeed page where public photos of all users can be seen

Requirements:-
Django==1.11.5

Installation:-

git clone https://github.com/SukhnandanMalhotra/PHOICS
